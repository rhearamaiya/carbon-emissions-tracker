About

  • Created to help users keep track of carbon emissions in hopes of lowering it
  • Uses Carbon Interface API for carbon emission output calculations of common activities (electricity, flights, shipping, driving)
  • Uses Flight API for flight departure/destination airport IATA codes
  • Made with Javascript, HTML, CSS, and Bootstrap

  • Near future developments:
